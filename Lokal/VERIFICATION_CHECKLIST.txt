TROUBLESHOOTING CHECKLIST
═══════════════════════════════════════════════════════════════════════════

If program won't start:

  ☐ Python installed? Run: python --version
  ☐ Dependencies installed? Run: pip list | grep mediapipe
  ☐ File exists? Check: dir Lokal\Vision*

If "Failed to connect to stream":

  ☐ Is ESP32P4 simulator running?
  ☐ Try test_stream.py to find correct URL
  ☐ When found, update STREAM_URL in Vision Detection.py

If "ModuleNotFoundError: mediapipe":

  ☐ Reinstall: pip install --upgrade mediapipe
  ☐ Or: pip install -r Lokal\requirements_vision.txt

If "No detections shown":

  ☐ Lower CONFIDENCE_THRESHOLD (try 0.3)
  ☐ Check stream is working (try test_stream.py)
  ☐ Try MODEL_SIZE = 's' for better accuracy

If "Very slow / low FPS":

  ☐ Increase TARGET_FPS (lower = faster)
  ☐ Use MODEL_SIZE = 'n'
  ☐ Close other applications
